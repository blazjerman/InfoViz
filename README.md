# Monthly Electricity Generation Explorer

An interactive R Shiny application for exploring monthly electricity generation by production source in European countries.

The project uses 2026 Eurostat data from:

https://ec.europa.eu/eurostat/databrowser/view/NRG_CB_PEM__custom_200961/default/table?lang=en

## Live app

[Monthly Electricity Generation Explorer](https://blazjerman.shinyapps.io/monthly-electricity-generation/)

## Features

- Searchable electricity generation data table
- Monthly total generation chart for a selected country
- Annual generation by selected production sources
- Europe map showing a source as a percentage of total generation
- Pie chart comparing one source between countries
